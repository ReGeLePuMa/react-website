# Răzvan Costin Tudose — academic portfolio

Modern React/Vite academic portfolio for razvantudose.ro.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The production files are generated in `dist/`.

## Deployment

For Netlify/Vercel/GitHub Pages-style hosting, deploy the repository with:

- build command: `npm run build`
- publish directory: `dist`

For manual upload, upload the contents of `dist/` to the hosting platform.
